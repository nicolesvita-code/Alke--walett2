# Alke Wallet 

Proyecto Front-end desarrollado para el módulo **Fundamentos del Desarrollo Front-end**.

 Descripción
Alke Wallet es una aplicación de billetera digital que permite a los usuarios gestionar sus activos financieros de forma simple y segura.  
La aplicación simula funcionalidades básicas como login, visualización de saldo, depósitos, envíos de dinero y registro de transacciones.

##  Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- jQuery

##  Funcionalidades
- Inicio de sesión de usuario
- Visualización de saldo disponible
- Depósito de fondos
- Envío de dinero a contactos
- Historial de transacciones
- Interfaz responsive
